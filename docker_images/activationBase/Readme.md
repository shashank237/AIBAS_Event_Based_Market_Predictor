Course Details  
Created for the course "M. Grum: Advanced AI-based Application Systems" by the Junior Chair for Business Information Science, esp. AI-based Application Systems at the University of Potsdam.

Image Purpose  
This Docker image provides a single activation dataset (`activation_data.csv`) that is used to trigger the AI system for inference.  
The file contains exactly one data instance taken from the test dataset to simulate an event-based market activation.

Data Origin  
The activation data is derived from the test dataset generated from publicly available financial market data sources.
