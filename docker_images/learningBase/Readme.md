Course Details  
Created for the course "M. Grum: Advanced AI-based Application Systems" by the Junior Chair for Business Information Science, esp. AI-based Application Systems at the University of Potsdam.

Image Purpose  
This Docker image provides the training and validation datasets used for the AI-based Event-Based Market Predictor.  
It contains the files `training_data.csv` and `test_data.csv`, which were created by scraping and preprocessing publicly available financial market data.

Data Origin  
The data was scraped from publicly accessible online financial market sources and processed for academic purposes only.
